About This Website

Features

  • Get detailed enrollment statistics for any UCI course from 2016 to the present:
    • Number of enrolled students
    • Max number of students
    • Number of students who requested to register
    • Number of students in the waitlist
    • Number of spots reserved for incoming freshmen/transfers
    • Course status (Open, Full, Waitlist, New Only)
  • Filter your searches based on instructor and/or course type
  • Detailed enrollment data for any type of UCI course (lectures, labs, discussions, etc.)
  • View enrollment data with organized graphs and tables
  • A search history menu that allows you to easily see the enrollment data of courses you searched before
  • View other quarters that offer a specific course

How do I use this website?

  1. Choose one specific course (e.g., WRITING 39C in Fall 2021). Here are some ways to select a course:
    • Select a Department, Course Number, and Quarter.
      • If you want to narrow down your search, you can add an Instructor and/or Course Type (the default is all course types).
    • If you only want to see one specific section of a course, you can just enter a Course Code and Quarter.
    • Type a Course Title from WebSoc, and select a Quarter.
      • If you want to narrow down your search, you can add an Instructor and/or Course Type (the default is all course types).
  2. When you're ready, click the Submit button. If you want to clear your whole search, press the Reset button.
  3. You will be directed to the enrollment data; to switch between tabs, click the Graphs, Tables, or Quarters buttons below the name of the course.
  4. To open/close graphs and tables, click the appropriate blue button on the right.
  5. To see the enrollment data of a course you searched before, click the History button and click the appropriate course.

What is the purpose of this website?

The purpose of this website is to help you plan out your schedule for future quarters! Many courses (especially general education courses and courses required for popular majors) fill up quickly, so it is nice to know when these classes have filled up in previous quarters. If a class is consistently popular, be sure to search for backup courses that do not fill up as easily. Overall, this website can help you determine the popularity of certain courses and instructors.

What quarters are currently being tracked?

Currently, 2021 Summer Session 2 and 2021 Fall Quarter are being regularly updated.

How much data is there?

I started collecting data in the end of July 2021, which means 2021 Summer Session 2 and 2021 Fall Quarter are the earliest quarters that have a substantial amount of enrollment data. Unfortunately, I don't know of a way to get the enrollment data of previous quarters; if anyone has data or a way to get this data, feel free to use the feedback form and inform me!

How often is data collected?

I collect data daily (normally in the morning). I will start collecting data for a certain quarter on the first day students can enroll in that quarter's classes. I will stop collecting data for a certain quarter on the last day students are able to freely register on WebReg.
